Maskable icon safe zone

For icon-192.png and icon-512.png (marked purpose: "any maskable"), the visible logo must fit within the center 80% of the canvas. Android clips icons into circles, squircles, or other shapes — anything outside the safe zone gets cut.

⚽

Gold background fills the full canvas. Icon stays within the dashed safe zone (80% center).

Recommended canvas spec

  • Background color: #854d0e (Gold 800 — matches PWA theme_color)
  • Safe zone: Center 80% — icon mark only, no text
  • Icon color: White or #fbbf24 on gold bg
  • Border-radius on canvas: 0px — OS applies its own clipping
  • Format: PNG-24 with no transparency for maskable

Export Guidelines

SVG requirements

  • Flatten all text to paths — no font embedding in SVGs
  • Remove invisible layers and artboard backgrounds
  • Optimize with SVGO (removes unnecessary attributes)
  • Use viewBox not fixed width/height on root
  • Illustrations: max 8kb after optimization
  • Icons: max 2kb after optimization
  • Dark mode adaptable: use currentColor for strokes/fills where possible

PNG requirements

  • Export at @2x resolution minimum (display size × 2)
  • Compress with TinyPNG or Squoosh — target <200kb for marketing images
  • PWA icons: PNG-24 (no transparency for maskable, transparent allowed for others)
  • OG images: JPEG quality 90 is acceptable to keep under 300kb
  • Template previews: PNG-24 at exactly 420×588px

Delivery format

  • Figma file with all assets on labeled frames
  • Exported to a zip: /svg/, /png/, /ico/ folders
  • File naming: lowercase, hyphens, no spaces (e.g. icon-192.png)
  • Template previews uploaded directly to Firebase Storage by developer
  • All other assets placed in public/ directory

Testing checklist

  • Favicons: verify in Chrome, Firefox, Safari, and Edge browser tabs
  • PWA icons: install app on Android and verify home screen icon
  • Apple touch: add to iOS home screen — check rounding and clarity
  • OG images: test with opengraph.xyz or Meta's debugger
  • Empty states: trigger each empty state in the app to verify sizing
  • Placeholders: create player/team with no photo and verify
Total estimated design effort: P1 assets (~2–3 hours for logo + icons + favicon pack using a generator tool + 2–3 placeholders). P2 assets (~1 day for illustrations + OG image + template previews). P3 assets (~1–2 days). The logo and favicons are the true blocker — everything else can ship as a second pass.
